## Making programs

Cloning the repo
```bash
$ git clone --recursive https://github.com/inozo22/fdf.git
```
To compile fdf

```bash
$ make
```

## How to use

Execute fdf

```bash
$ ./fdf ./test_maps/42.fdf
```
